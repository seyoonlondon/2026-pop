# 📊 지역별 연령별 인구 구조 뷰어

행정안전부 주민등록 연령별 인구현황 데이터를 활용해, 특정 지역의 연령별 인구 분포를 시각화하고 전국(읍면동 포함)에서 인구구조가 가장 비슷한 지역 Top 5를 찾아주는 Streamlit 웹앱입니다.

## 주요 기능

- **지역 선택**: 검색창에 지역명 일부를 입력해 목록을 좁히거나, 검색 없이 전체 목록에서 바로 선택 가능 (시/도, 시/군/구, 읍/면/동 모두 포함)
- **연령별 인구 분포 그래프**: 선택한 지역의 0세~100세 이상 인구를 성별(계/남/여)별 꺾은선 그래프로 표시 (Plotly)
- **인구구조 유사 지역 Top 5**: 선택한 지역과 전국 지역들의 연령별 인구 *비율(%)*을 코사인 유사도로 비교해, 인구구조(연령 분포의 모양)가 가장 비슷한 지역 5곳을 찾아 함께 그래프로 표시
- 총 인구수, 원본 데이터 확인 등 부가 정보 제공

## 파일 구성

```
├── app.py                                    # Streamlit 앱 코드
├── requirements.txt                          # 의존 패키지 목록
└── 202606_202606_연령별인구현황_월간.csv        # 원본 데이터 (같은 폴더에 위치해야 함)
```

## 데이터 출처 및 형식

- 파일: `202606_202606_연령별인구현황_월간.csv`
- 인코딩: `cp949` (EUC-KR 계열)
- 구조: 행정구역별로 `{연월}_{성별(계/남/여)}_{나이(0세~99세, 100세 이상)}` 형식의 컬럼 310개 포함
- 숫자 컬럼은 천 단위 콤마(`,`)가 포함된 문자열이므로 앱 내부에서 자동으로 숫자형으로 변환됨

> 데이터 파일이 앱과 다른 이름이거나 다른 달의 데이터로 교체되는 경우, `app.py` 상단의 `DATA_FILE` 값만 파일명에 맞게 수정하면 됩니다 (연월 컬럼명은 자동으로 감지됩니다).

## Streamlit Cloud 배포 방법

1. `app.py`, `requirements.txt`, 원본 CSV 파일 세 개를 같은 GitHub 저장소에 커밋합니다.
2. [share.streamlit.io](https://share.streamlit.io) 에 접속해 로그인합니다.
3. "New app"에서 해당 저장소와 브랜치, `app.py` 경로를 지정합니다.
4. Deploy를 누르면 자동으로 빌드 후 배포됩니다.

## 유사 지역 산출 방식

1. 각 지역의 연령별 인구수를 해당 지역 총인구수로 나눠 **연령대별 인구 비율(%)** 로 정규화합니다 (인구 규모 차이를 배제하고 구조의 "모양"만 비교하기 위함).
2. 정규화된 벡터 간 **코사인 유사도**를 계산합니다.
3. 인구수가 0인 지역은 비교 대상에서 제외합니다.
4. 선택한 지역 자신을 제외하고 유사도가 가장 높은 상위 5개 지역을 추출합니다.

## 화면 구성 요약

1. 지역 검색 + 선택, 성별 선택
2. 선택 지역 총 인구수
3. 선택 지역의 연령별 인구 분포 꺾은선 그래프
4. 전국 인구구조 유사 지역 Top 5 그래프 + 유사도 표
5. 원본 데이터 확인 (펼쳐보기)
